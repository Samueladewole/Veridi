Security Architecture Matrix
🌐 Cloudflare Edge
WAF — OWASP Core + custom Veridi rules
DDoS — L3/L4/L7 auto-mitigation
Rate limiting — per key, per IP, per endpoint
Bot scoring — challenge <50 score requests
mTLS — Worker ↔ Railway internal comms
🔑 API Authentication
API keys — SHA-256 hashed, never stored plain
JWT dashboard sessions — 15min access / 7d refresh
HMAC-SHA256 — all outbound webhook payloads signed
Idempotency keys — safe to retry, no double-charge
Consent token — required on every verification call
🗃️ Data Protection
PII tokenisation — NIN/BVN → Argon2id hash only
AES-256 — field encryption for sensitive columns
Raw biometrics — never persisted, deleted post-match
Audit log — immutable append-only access trail
NDPA 2023 — consent records, right to erasure API
🚂 Railway Hardening
No public Railway URLs exposed — all via Cloudflare
Env vars — all secrets in Railway environment, no .env files
Private networking — services communicate internally only
Docker — no shell access in production containers
Health checks — auto-restart on failure, zero-downtime deploy
⚠️ Abuse Prevention
Consent token expiry — 10 min, signed, one-time use
Verification dedup — same NIN + client: serve cache
Suspicious pattern detection — flag rapid sequential calls
IP allowlisting — enterprise clients can lock to IPs
Client spend caps — configurable max monthly spend limit
📊 Observability
Sentry — error tracking with PII scrubbing enabled
Railway Metrics — CPU, memory, request rate per service
Cloudflare Analytics — edge request data, attack logs
Custom dashboards — latency p50/p95/p99 per endpoint
Alerting — PagerDuty on p99 >2s or error rate >1%
